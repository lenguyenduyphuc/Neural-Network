# Neural Networks: Zero to Hero
## By Andrej Karpathy

A practical and code-first introduction to deep learning, neural networks, and PyTorch.

### Course Overview

This course takes you on a journey from the fundamentals of neural networks to implementing state-of-the-art architectures from scratch. The focus is on building intuition and practical understanding through hands-on coding.

### Prerequisites

- Basic Python programming
- Understanding of basic mathematics (algebra, calculus helpful but not required)
- No prior machine learning experience needed

### Environment Setup

```bash
# Required packages
pip install torch numpy matplotlib jupyter
```

### Course Content

1. **The Spelled-Out Intro to Neural Networks and Backpropagation**
   - Building micrograd from scratch
   - Understanding automatic differentiation
   - Basic neural network implementation

2. **makemore: Building makemore**
   - Character-level language modeling
   - Bigram model
   - Neural network implementation

3. **Building GPT**
   - Attention mechanism
   - Transformer architecture
   - Training your own mini-GPT

4. **Backpropagation**
   - Detailed walkthrough of backpropagation
   - Manual gradient calculations
   - Understanding the chain rule
